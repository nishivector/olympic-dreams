# Olympic Dreams

An absurd Olympic sports competition game with three wild events!

## How to Play

- **Slalom**: Ski through gates, avoid missing them!
- **Javelin**: Hold SPACE to charge, release to throw
- **Diving**: Time your dive for a perfect entry!
